# Information Architecture (Current Desktop)

当前桌面端左侧 TAB 信息架构（按用户使用频率排序）：

1. `更新收益`（快捷录入弹窗入口）
2. `财富总览`
3. `投资收益`
4. `FIRE进度`
5. `收入分析`
6. `消费分析`
7. `导入中心`
8. `高级管理`

## 结构说明

- `更新收益`：不是页面 TAB，而是快捷录入动作入口。
- `财富总览`：财富结构（Sankey）+ 趋势（堆叠图）+ 核心指标。
- `投资收益`：投资收益表现 + 账户收益率对比。
- `FIRE进度`：FIRE 指标、预算概览与复盘、预算项管理。
- `消费分析`：消费总览（数据渲染）+ 交易分析相关操作。
- `导入中心`：三类导入器 + 规则管理。
- `高级管理`：数据库健康、查询维护、管理员工具；部分功能受开发者模式控制。

## UI 约定（关键）

- 页签顶部标题栏提供主信息；页内重复标题尽量减少。
- 查询类模块默认自动刷新，减少“查询按钮”。
- 复杂图表优先数据渲染，不再走内嵌 HTML。
