# Product Flow (Current Desktop)

本文件描述 **当前 Tauri Desktop** 的实际产品主流程（精简版）。

## 主流程 1：导入 -> 校验 -> 分析

1. 打开 `导入中心`
2. 选择导入来源（YZXY / 招行 EML / 招行 PDF）
3. 先执行 `预览`
4. 确认摘要无异常后执行 `导入`
5. 切换到 `财富总览 / 投资收益 / 消费分析` 查看结果

## 主流程 2：更新收益（快捷入口）

1. 点击左侧 `更新收益`
2. 弹出快捷录入窗口（默认今日 + 上次账户）
3. 填写总资产与净转入/转出
4. 提交后自动刷新相关分析页面

## 主流程 3：规则优化（导入质量提升）

1. 在 `导入中心 -> 规则管理` 查看商户建议
2. 回填 `商户映射/分类规则/白名单/排除规则`
3. 返回导入器重新 `预览`
4. 确认分类质量提升后再导入

## 主流程 4：高级管理（仅必要时）

- 数据库健康检查
- 查询与维护（账户元数据、投资记录、资产估值、账户目录）
- 管理员重置/开发者模式能力（按需）

## 当前设计原则

- 默认路径以“可用性”为主，不暴露过多调试细节
- 高风险操作收敛在 `高级管理`
- 导入链路先 `预览` 再 `导入`
- 所有金额显示受全局隐私开关控制
