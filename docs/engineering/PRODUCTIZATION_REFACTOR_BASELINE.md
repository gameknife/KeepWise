# Productization Refactor Baseline (Current)

本文件记录桌面端 UI 从“迁移验证页”向“产品工作台”收敛时的基线原则。

## 已完成的收口方向

- 左侧 TAB 工作台结构（产品入口）
- 调试能力向 `高级管理` 收敛
- 消费总览从内嵌 HTML 改为数据渲染
- 图表风格统一（轻量化、深色主题、隐私遮罩联动）
- 设置系统与隐私开关落地（本地持久化）

## 当前仍在进行的收口点

- 页内重复标题与描述继续精简
- 查询/维护模块的交互一致性（自动刷新、行内操作、弹窗）
- 非开发者视角文案与空状态优化

## UI 设计约定（当前有效）

- 页签标题栏负责主信息，页内重复标题尽量减少
- 自动查询优先，保留写操作按钮
- 高风险/调试能力默认隐藏或收敛
- 金额显示统一受隐私开关控制
