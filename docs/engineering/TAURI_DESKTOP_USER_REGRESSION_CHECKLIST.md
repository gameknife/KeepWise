# Desktop User Regression Checklist (Concise)

用于 RC 或重大 UI/逻辑变更后的人工回归（按左侧 TAB）。

## 通用检查

- 应用启动正常（默认进入 `财富总览`）
- 侧栏收纳/展开正常
- 隐私按钮与设置按钮可用
- 金额隐私遮罩生效
- 深色主题下下拉与滚动条可读（含 Windows）

## 更新收益（快捷录入）

- 点击侧栏按钮弹出录入窗口（不切页）
- 默认日期为今天
- 默认账户为上次录入账户
- 提交后相关分析数据刷新

## 财富总览

- Sankey 财富结构图正常
- 堆叠趋势图正常（负债在负轴）
- 资产类型筛选联动正确
- `相比<n月m日>` 指标口径正确

## 投资收益

- 投资收益表现（收益率+曲线）正常
- 账户收益率对比正常
- 预设区间默认 `年初至今`

## FIRE进度

- `自由度` 重点卡显示正常
- 预算概览与复盘联动正常
- 预算项管理（列表/新建弹窗/删除）正常

## 收入分析 / 消费分析

- 收入概览按年切换正常
- 消费总览筛选（月份/分类/商户）联动正常
- 消费图表/tooltip/隐私遮罩正常

## 导入中心

- YZXY / EML / PDF 均能预览和导入
- 规则管理查询/写入/删除可用
- 商户建议回填可用

## 高级管理

- 管理员数据库健康可用
- 查询维护（元数据/投资/资产估值/账户目录）可用
- 投资记录行内修正/删除可用
- 开发者模式切换正常（如本次回归涉及）
